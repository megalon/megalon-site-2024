Multiplayer Graffiti Game
2024

A multiplayer painting game for PC with persistent worlds!

Features:

  • See what others painted in a world before you
  • Paint on everything, including your friends
  • Accurate painting sync
  • Pen tablet support
  • Freecam in a seprate window for streaming / recording at a distance

Built in Unity using Network for Game Objects (NGO). This project is designed to solve many of the issues I had making the VR Chat world Graffiti Walls!
