This Website
2024

I made this website. I did not use a site generator or any front-end libraries like React of Vue.

I decided to write my own page loader to learn how they work.

goals

  • A simple, single page site that loads quickly
  • Fast iteration when editing (hot-reload)
  • Reactive CSS for small and large screen sizes
  • Light and Dark themes that match system settings
  • Easy to understand URLs

development

The site is built using Typescript, SCSS, and HTML. I used Vite to handle builds and hot-reloading.

There is only one main page, but the data for each project "page" is split into separate HTML files. Often when navigating a website the whole page reloads when the URL changes, and this can cause the page to flash white in some browsers. I am able to avoid this by loading the data for each project into the main page, even though the URL changes.

styling

This design of this website is inspired by '70s instruction manuals. Simple, stark, monospaced text on a plain background.

SCSS allowed me to create reusable variables for the colors on the page, which drastically simplified the light and dark themes.

deployment

The site is run from a small Linux server that I administer. It is served with NGINX and routed through Cloudflare.

In the future I would like to create automatic builds of a Docker container on GitHub, and auto deployment to the server. In practice this feels unnecessary at this time.
